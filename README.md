Онлайн-магазин одежды
Описание проекта:
Данный проект представляет собой веб-приложение онлайн-магазина одежды. В качестве хранилища данных используется LocalStorage, что позволяет сохранять состояние корзины и выбранных товаров без серверной части.

Проект адаптирован под различные устройства и поддерживает мобильную версию.

Основной функционал
- Добавление товаров в корзину
- Управление корзиной покупок
- Примерка одежды на манекене (виртуальная примерка)
- Интерактивная карта (Яндекс Карты)
- Сохранение данных через LocalStorage
- Добротная верстка с дополнительными мелкими интерактивными функциями
- Адаптивный дизайн под мобильные и десктоп устройства
  
Страницы проекта:
1. Главная страница
Общая информация о магазине
Навигация по сайту
Презентация основных категорий
2. Страница со всей одеждой
Полный каталог товаров
Фильтрация и просмотр ассортимента
Быстрое добавление товаров в корзину
3. Страница товара
Детальное описание одежды
Просмотр изображений
Возможность добавить в корзину
Переход к примерке
4. Примерочная с корзиной
Виртуальная примерка одежды на манекене
Управление выбранными товарами
Просмотр и редактирование корзины
Подготовка к покупке

Технологии:
HTML
CSS (адаптивная верстка)
JavaScript
LocalStorage
Яндекс Карты API

Цель проекта:
Создать удобный и интерактивный онлайн-магазин одежды с возможностью примерки вещей и хранения данных без серверной части, используя возможности браузера.
